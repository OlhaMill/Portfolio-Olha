On this page, you will find a series of projects that I have developed as part of my training and practice in the field of software development. These projects are an introduction to my skills and abilities and represent the first step in my path as a developer.
Each project presented here is a demonstration of my basic knowledge and enthusiasm for learning new technologies. Although they are simple projects, they are designed to showcase my ability to apply fundamental concepts, solve problems, and build functional solutions.

This project arose from my recognition of the crucial need for nutritionists to have a tool that allows them to effortlessly create personalized diets, track each client comprehensively, and manage their own database of diets. This was my final project for the Multiplatform Application Development (DAM) course, where I was able to apply and expand my knowledge in application development, database management, and user interface design.

The application was primarily developed using the Java programming language. For its architecture, the frontend was created with JavaFX, providing an interactive and modern user interface. The backend is built in Java 17, ensuring robustness and efficiency in process and data handling. Additionally, the REST API was implemented using Spring Boot, facilitating integration and communication with other services, ensuring optimal performance and a scalable structure.

Through this project, I learned the importance of flexibility and adaptability in software development, especially in a dynamic field such as nutrition and health. The effort was worth it, as I received an honorable mention for the project, highlighting the level of dedication and quality of the work done.

The Marvel Heroes project originated as part of an assignment to create a client application for a REST API developed in JavaScript (Node.js), which interacts with a database in MongoDB. The client application was implemented using JavaFX.

The main goal of this project was to demonstrate the internal functionality of the REST API, allowing users to interact with the database effectively and access information related to Marvel heroes. The application provides an intuitive graphical interface that facilitates interaction with the API and visualization of the stored data.

This project showcases my ability to integrate backend and frontend technologies, manage non-relational databases, and create dynamic and functional user interfaces.

ViajaYA is an application developed in Kotlin using Android Studio as the development environment, as the final project for a course in the Multiplatform Application Development (DAM) program. The app is designed to meet the basic needs of a traveler, offering features such as destination search, creation and management of personalized itineraries, booking of accommodations and transportation, and the ability to track expenses during the trip.

For data management and user authentication, the application connects to Firebase, providing a secure and real-time environment. The design is intuitive and user-friendly, aimed at facilitating quick and efficient travel planning.
