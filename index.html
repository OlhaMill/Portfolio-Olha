En esta página, encontrarás una serie de proyectos que he desarrollado como parte de mi formación y práctica en el campo del desarrollo de software. Estos proyectos son una introducción a mis habilidades y capacidades, y representan el primer paso en mi camino como desarrolladora.
Cada proyecto aquí presentado es una demostración de mis conocimientos básicos y mi entusiasmo por aprender nuevas tecnologías. Aunque son proyectos sencillos, están diseñados para mostrar mi habilidad para aplicar conceptos fundamentales, resolver problemas y construir soluciones funcionales.

Este proyecto surge de mi reconocimiento de la necesidad crucial de que los nutricionistas tengan una herramienta que les permita crear dietas personalizadas sin esfuerzo, realizar un seguimiento completo de cada cliente y gestionar su propia base de datos de dietas. Este ha sido mi proyecto final para el ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM), en el que he podido aplicar y expandir mis conocimientos en desarrollo de aplicaciones, gestión de bases de datos y diseño de interfaces de usuario.

La aplicación se ha desarrollado principalmente utilizando el lenguaje de programación Java. Para su arquitectura, el frontend se ha creado con JavaFX, proporcionando una interfaz de usuario interactiva y moderna. El backend está construido en Java 17, garantizando robustez y eficiencia en el manejo de procesos y datos. Además, la API REST se ha implementado utilizando Spring Boot, lo que facilita la integración y comunicación con otros servicios, asegurando un rendimiento óptimo y una estructura escalable.

A través de este proyecto, he aprendido la importancia de la flexibilidad y la adaptabilidad en el desarrollo de software, especialmente en un campo tan dinámico como la nutrición y la salud. El esfuerzo ha valido la pena, ya que he obtenido una mención honorífica por el proyecto, lo cual destaca el nivel de dedicación y la calidad del trabajo realizado.

El proyecto Héroes Marvel se originó como parte de una tarea destinada a la creación de una aplicación cliente para una API REST desarrollada en JavaScript (Node.js), la cual interactúa con una base de datos en MongoDB. La aplicación cliente fue implementada utilizando JavaFX.

El objetivo principal de este proyecto fue mostrar la funcionalidad interna de la API REST, permitiendo a los usuarios interactuar con la base de datos de manera efectiva y acceder a la información relacionada con los héroes de Marvel. La aplicación proporciona una interfaz gráfica intuitiva, que facilita la interacción con la API y la visualización de los datos almacenados.

Este proyecto demuestra mi capacidad para integrar tecnologías de backend y frontend, manejando bases de datos no relacionales y creando interfaces de usuario dinámicas y funcionales.

ViajaYA es una aplicación desarrollada en Kotlin utilizando Android Studio como entorno de desarrollo, como proyecto final de una asignatura del ciclo formativo de Desarrollo de Aplicaciones Multiplataforma (DAM). La app está diseñada para satisfacer las necesidades básicas de un viajero, ofreciendo funcionalidades como la búsqueda de destinos, la creación y gestión de itinerarios personalizados, la reserva de alojamientos y transporte, y la posibilidad de llevar un seguimiento de gastos durante el viaje.

Para la gestión de datos y autenticación de usuarios, la aplicación se conecta a Firebase, proporcionando un entorno seguro y en tiempo real. El diseño es intuitivo y accesible, pensado para facilitar la planificación de viajes de manera rápida y eficiente.
